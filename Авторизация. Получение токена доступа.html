Для работы с сервисом авторизации в GET запросе используется Basic авторизация.

(Получить код авторизации необходимо у ответственного сотрудника компании «Стиль Д»)

 

GET

https://api.prostor.ua/api/hs/AuthService/v1/login

Request headers

Название

Тип

Описание

Authorization

String

Authorization header

Допустимые значения: “Basic value”. Например:

Basic YWhhbWlsdG9uQGFwaWdlZS5jb206bXlwYXNzdzByZAo

papiauth

string(36)

Для клиента мобильного приложения. Ключ, который использовался при ОТП авторизации конкретного пользователя в заголовке papiauth. Предназначен для идентификации клиента.

 

Ответ сервера (Success 200)

Название

Тип

Описание

Success

boolean

Результат запроса.

“true” - успешный запрос

“false” - ошибка

Response/KEY

String(36)

Ключ доступа к сервису получения баланса (ограничен по времени)

Error

String

Описание ошибки.

Пример запроса/ответа сервиса: 

Запрос:

https://api.prostor.ua/api/hs/AuthService

    "Host": "api.prostor.ua",

    "Cache-Control": "no-cache",

    "Connection": "close",

    "Accept-Encoding": "gzip, deflate, br",

    "Accept": "*/*",

    "Authorization": "Basic YWhhbWlsdG9uQGFwaWdlZS5jb206bXlwYXNzdzByZAo ",

    "Content-Length": "0"

Ответ:

    "Content-type": "application/json; charset=utf-8"

{

    "success": "true",

    "response": {

        "KEY": "f6595522-dfa6-407b-b15f-7cb4ae18ebbb"

    },

    "error": ""

}

Пример запроса с использованием CURL:

curl --location --request GET 'https://api.prostor.ua/api/hs/AuthService/v1/login' \

--header 'Authorization: Basic cG9saWNhcmQ6S3JlY2hldG92aWNo' \  --data-raw ''
